# keju-examiner

中文 | [English](README_EN.md)

这里上传了我在不想学习时收集的清代科举考官数据，抄录自[《清秘述闻》](https://www.shidianguji.com/book/CADAL11202250/chapter/1lk09dczg8sun?page_from=searching_page&version=3)和[《清秘述闻续》](https://www.shidianguji.com/book/CADAL11202250/chapter/1lk09dczg8sun?page_from=searching_page&version=3)，用DeepSeek简单提取了一些变量。供有需要的朋友取用，见[data](./data)

我一直想研究科举的考官选任。实际上，做一次乡试或会试的考官是一件收益远大于成本的工作，一次考试后，中榜的学生会成为你的“门生故吏”，对未来的仕途有很好的助力。在这种背景下，谁被选为考官就是一件很有趣的事，各方势力会为之博弈吗？皇帝会平衡吗？不过我始终没想到很好的实证切入点，就先这么放着吧
